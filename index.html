Taxi Man 4

Click here for a working demo.

Notes:

⋅ Built upon Backbone.JS framework

⋅ Game app that allows a player to first choose a vehicle, Corolla or Land Cruiser, which player can then control with arrow keys to move around and pick up passengers.

⋅ Every time the player moves, they lose fuel. 3 fuel points lost if Land Cruiser is chosen and 1 fuel point lost if Corolla is chosen.

⋅ Each time a passenger is picked up, a new passenger is generated at a random location on the grid.

⋅ Included a JavaScript library, MouseTrap, to allow usage of arrow keys instead of having to click buttons.
